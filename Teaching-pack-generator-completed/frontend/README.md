# Frontend - Teaching Pack Generator

This folder contains the web UI for the Teaching Pack Generator.

## Requirements

- Node.js 18+
- npm or yarn

## Setup

```bash
cd frontend
npm install
```

## Development Server

```bash
npm run dev
```

The app will be available at http://localhost:5173

## Other Commands

```bash
npm run build
npm run preview
npm run lint
```

## Folder Structure

```
frontend/
  src/
    components/
    types/
    App.tsx
    main.tsx
  public/
  index.html
```

## Tech Stack

- React 19.2
- TypeScript 5.9
- Vite 7.2
- Tailwind CSS 4.1
- ESLint
